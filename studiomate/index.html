Quick answers

  • Cancel a running sync? Click Cancel sync.
  • What does “Mirror: delete extra items in the target folder” do? It removes items that exist only in the target. Use with care.
  • Auto-run doesn’t start immediately? It triggers after a sync completes and the set interval elapses.
  • Progress looks stuck at 0%? Deletions/metadata updates can move the bar slowly; completion is accurate.
